HTML Notes

  • HTML stands for HyperText Markup Language
  • It is used to create the structure of a website
  • It uses tags to define different elements on a webpage
  • Some common tags include <h1> for headings, <p> for paragraphs, and <a> for links
  • HTML is not a programming language, it is a markup language

CSS Notes

  • CSS stands for Cascading Style Sheets
  • It is used to style the appearance of a website
  • It uses selectors to target specific elements on a webpage and apply styles to them
  • Some common properties include color, font-size, and background-color
  • CSS can be written in an external stylesheet or within the HTML file using the <style> tag

JavaScript Notes

  • JavaScript is a programming language that is used to make websites interactive
  • It can be used to manipulate the HTML and CSS of a webpage
  • JavaScript code is written between <script> tags
  • Some common JavaScript concepts include variables, functions, and event handling
